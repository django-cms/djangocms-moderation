.. _comment:

Comments
========

Review is a conversation, so comments can be attached at two levels:

* the :term:`Moderation Collection` — e.g. notes about the batch as a
  whole, reachable via the comment icon in the collection changelist;
* each :term:`Moderation Request` — e.g. review feedback on a specific
  draft, reachable via the comment icon in the requests changelist.

In addition, a reviewer rejecting a request can attach a message to the
action itself, which is included in the rework notification sent to the
author.

Comments are visible to the collection author and to anyone involved in
moderating the collection. Either comment level can be switched off with
the ``CMS_MODERATION_COLLECTION_COMMENTS_ENABLED`` and
``CMS_MODERATION_REQUEST_COMMENTS_ENABLED`` settings — see :ref:`settings`.
