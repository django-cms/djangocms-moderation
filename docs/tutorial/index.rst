.. _tutorial:

Tutorial
========

New to django CMS Moderation? This two-part tutorial takes you from nothing
to a working editorial approval process.

#. :ref:`tutorial-quickstart` gets moderation running and publishes your
   first page through it — playing every part yourself, in about ten minutes.
#. :ref:`tutorial-review-process` builds on that same project to introduce a
   real review process: a separate reviewer, group permissions, rework and
   the rest of the feature set.

Work through them in order — part two assumes the project you create in part
one. You should be comfortable with the Django admin and have Python 3.10+
installed.
